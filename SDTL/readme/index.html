Synchornous Distributed Traning Library (PyTorch)¶

Contained within this library are multiple synchronous distributed training algorithms. Researchers who implement DistributedDataParallel in PyTorch need only replace a single line of code to utilize this library. The codes are available at Distributed Traning Library.

Vanilla PyTorch (DataParallel)¶

Use DistributedDataParallel on model. 

model = DistributedDataParallel(model, device_ids=[args.gpu])
model.forward(data); model.backward(loss) ; optimizer.step()

SDTL (LSGD, LSGD+, EASGD, EASGD+)¶

Use CommOptimizer on local optimizer. 

import CommOptimizer
optimizer = CommOptimizer(optimizer, dist_optim_name='LSGD+', comm_period=1, dist_pulling_strength=0.1, local_pulling_strength=0.1)
model.forward(data); model.backward(loss) ; optimizer.step()

For more details of EASGD and LSGD, refer to [Leader Stochastic Gradient Descent Blog].
